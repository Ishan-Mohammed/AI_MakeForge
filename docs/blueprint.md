# **App Name**: MindForge AI

## Core Features:

- Multimodal Study Ingest: Comprehensive input terminal for pasting lesson content or uploading PDF, DOCX, and TXT files with real-time character counting.
- Neural Knowledge Synthesis: Advanced AI tool using Gemini to process raw content into a structured JSON schema comprising flashcards, concepts, notes, and definitions.
- 3D Flashcard Laboratory: Interactive study deck featuring high-fidelity flip animations, session navigation controls, and 'one-click copy' for individual cards.
- Adaptive Assessment Engine: An AI tool that generates varied testing tiers including MCQs, short answer, and complex essay questions to validate material mastery.
- Knowledge Graph Dashboard: A glassmorphic tabbed interface providing seamless navigation between concepts, bulleted revision notes, and technical definitions.
- Forge Export Hub: On-demand PDF synthesis for local offline study, featuring custom branding, professional typography, and formatted summary sections.
- Intelligent Study Analytics: Real-time metrics dashboard calculating extraction statistics and dynamic reading time estimates for the generated curriculum.

## Style Guidelines:

- A sophisticated 'Interstellar Neural' dark palette featuring Primary Electric Violet (#8B5CF6) for vibrancy, Deep Obsidian Background (#0B0A0F) for depth, and Ice Indigo Accent (#A5B4FC) for highlighting and navigation.
- A tech-forward font pairing: 'Space Grotesk' for bold, scientific headlines and 'Inter' for highly legible, objective body text. 'Source Code Pro' is used for monospaced data fields.
- Lucide-style linear icons with ultra-thin weights and subtle gradients, evoking a premium 'futuristic digital ecosystem' aesthetic.
- A layout driven by high-end glassmorphism principles: background-blur-heavy sidebars, floating dashboard containers, and staggered masonry grids for concept tags.
- A premium interaction layer featuring 'Framer Motion' transitions, magnetic button effects, and a flowing neural pathway background animation.
